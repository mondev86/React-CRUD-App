# Modern React Study (CRUD)

Proyecto CRUD minimo para estudiar React actual sin ruido.

## Requisitos

- Node.js 20+ (recomendado)

## Ejecutar

```bash
npm install
npm run dev
```

## Estructura corta

- `src/main.jsx`: punto de entrada.
- `src/App.jsx`: logica CRUD (crear, listar, editar y eliminar).
- `src/index.css`: estilos globales minimos.

## Funcionalidades

- Crear registros.
- Listar registros.
- Editar registros.
- Eliminar registros.
